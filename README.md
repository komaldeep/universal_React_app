# Universal React app
### How to run the map

 * Open your Terminal
 * git clone "https://github.com/komaldeep/universal_React_app"
 * npm install
 * npm run start-dev
 * open the http://localhost:3000/ and see the magic
 
### Tech stack
 * React
 * Redux
 * React- router
 * Express
 * Node.js
 * webpack
